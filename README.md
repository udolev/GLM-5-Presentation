# GLM-5 Technical Report Presentation

A short presentation based on the GLM-5 technical report, [GLM-5: from Vibe Coding to Agentic Engineering](https://arxiv.org/abs/2602.15763).

The deck was created together with Codex as HTML slides. The GLM-5 paper source and BibTeX were cloned into [`GLM-5-arXiv`](GLM-5-arXiv), then used as the working reference while we went section by section and refined the technical explanations.

Open [`slides/index.html`](slides/index.html) to view the presentation locally.
